# 🧠 Finance Simulation Platform

A multi-agent, ML-powered investment simulator built in collaboration with **Dell Technologies** as part of a capstone project. This interactive system offers personalized stock recommendations based on user preferences, news sentiment, and Monte Carlo risk modeling.

---

## 🚀 Project Overview

Retail investors often lack intelligent tools that can simulate real-world investment behavior based on personal preferences and external factors. Our platform addresses this gap by combining:

- 🧮 Machine Learning for stock direction prediction
- 📰 Real-time News Sentiment Analysis
- 🎲 Monte Carlo Simulations for portfolio risk forecasting
- 🧍 Multi-agent design to model investor behavior, market context, and news impact

---

## 🔧 Features

- **Buy/Sell Recommendations** via XGBoost & Random Forest classifiers
- **Sentiment-aware Decisioning** with NLP (TextBlob/VADER)
- **Personalized Simulations** based on risk appetite, capital, and horizon
- **Monte Carlo Portfolio Forecasting** using Geometric Brownian Motion
- **Modular Multi-Agent Architecture**: User, Recommendation, News, Portfolio Manager


---

## 📊 Technologies & Libraries

- Python 3.9+
- scikit-learn, XGBoost, pandas, NumPy
- VADER, TextBlob (for sentiment analysis)
- Matplotlib, Seaborn
- Streamlit (optional frontend)
- Yahoo Finance API or yfinance (for stock data)
